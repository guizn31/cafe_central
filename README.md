Objetivo do site Café Central

 Divulgar seus produtos e facilitar o contato com os clientes por meio de um site institucional autoral para assim fortalecer a identidade de sua marca.



Público-alvo
Todas as idades.

Páginas institucionais definidas:
1. Menu
2. Cardápio
3. Chat com o cliente
4.Feedback 
5.Sobre nós

Requisitos Funcionais
Exibir localização da loja
Exibe todos os produtos oferecido pelo cardápio da loja
Disponibilizar contato com a loja
Garantir navegação funcional entre páginas


2- Requisitos Não Funcionais
Layout organizado e intuitivo
Código HTML semântico, legível e indentado
Compatibilidade com navegadores modernos
Compatibilidade com múltiplas plataformas

3- Requisitos Estruturais
Menu de navegação presente em todas as páginas
Estrutura de pastas padronizada


Organização do ambiente de desenvolvimento:
IDE utilizada: Visual Studio Code
Pasta raiz do projeto criada: Cafe-central
Padrão de nomenclatura aplicado (letras minúsculas, sem espaços)
Projeto preparado para separação entre HTML, CSS e JavaScript

Estrutura técnica definida e organizada:
Diretórios criados:
/css 3 arquivos de estilo (layout e identidade visual)
/js 3 scripts JavaScript (interações futuras)
/img 3 imagens dos produtos da loja
/pages 3 páginas internas do site
